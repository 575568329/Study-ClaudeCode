# 🚀 欢迎使用ClaudeCode学习项目

这是一个系统化的学习项目，帮助你全面掌握ClaudeCode的能力。

## 📚 项目特点

- **🤖 AI驱动学习**：Claude Code担任私人技术导师
- **📊 全面进度跟踪**：100个技能点的详细掌握情况
- **🧠 Obsidian知识管理**：双向链接、图谱视图、标签系统
- **📝 每日学习会话**：完整的学习历程记录

## 💡 使用建议

### 日常学习流程

1. **准备阶段**
   - 打开Obsidian查看今日会话笔记
   - 回顾上次学习内容

2. **学习阶段**
   - 启动Claude Code
   - 提问 → 解释 → 理解检查
   - 实践验证

3. **记录阶段**
   - Claude自动更新会话笔记
   - 更新技能跟踪
   - 创建知识库笔记

4. **复习阶段**
   - 查看知识图谱
   - 检查反向链接
   - 规划下次学习

### 学习原则

- **持续一致**：每日学习，积少成多
- **主动思考**：苏格拉底教学法，多问为什么
- **实践验证**：理论结合实际操作
- **知识关联**：充分利用双向链接
- **定期复习**：通过图谱回顾知识网络

## 🎓 学习方法

### 苏格拉底教学法

Claude会：
1. 先问你："你已经了解多少？"
2. 基于你的现有知识构建解释
3. 通过提问检查你的理解
4. 根据反馈调整教学方式

### 知识管理

- **双向链接**：连接相关概念和会话
- **标签系统**：多维度组织知识
- **图谱视图**：可视化知识网络
- **MOC笔记**：构建内容地图
